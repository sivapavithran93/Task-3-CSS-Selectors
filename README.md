# Task 3 - HTML & CSS Selectors

## Overview

This project demonstrates the use of different CSS selectors to style HTML elements. It includes examples of element selectors, class selectors, ID selectors, pseudo-class selectors, and the `nth-child()` selector.

## Website
link: https://sivapavithran93.github.io/Task-3-CSS-Selectors/

## Features

* Styled headings using a common CSS class.
* Applied a background color using an ID selector.
* Customized text colors using class and ID selectors.
* Styled paragraph elements using an element selector.
* Highlighted links in unordered lists using `:first-of-type` and `:last-of-type`.
* Applied different colors to specific ordered list items using `:nth-child()`.

## Technologies Used

* HTML5
* CSS3

## CSS Selectors Demonstrated

### Element Selector

* `p` selector used to style all paragraph elements.

### Class Selector

* `.head` used to style headings.
* `.bold` used to style bold text.
* `.para1` used to style text inside a paragraph.

### ID Selector

* `#bglime` used to apply a lime background color.
* `#para` used to customize a specific paragraph.

### Pseudo-Class Selectors

* `ul:first-of-type` styles the first unordered list.
* `ul:last-of-type` styles the last unordered list.

### nth-child Selector

* `ol li:nth-child(2)` styles the second list item.
* `ol li:nth-child(3)` styles the third list item.
* `ol li:nth-child(4)` styles the fourth list item.

## Learning Outcomes

Through this task, I learned how to:

* Apply CSS using different selector types.
* Target specific HTML elements efficiently.
* Use pseudo-class selectors to style elements based on their position.
* Customize individual list items using the `nth-child()` selector.
* Separate content (HTML) from presentation (CSS) for better code organization.
